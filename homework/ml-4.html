                      probabilities_1 = [0.83, 0.78, 0.62, 0.48, 0.32, 0.22]
                      probabilities_2 = [0.92, 0.62, 0.52, 0.49, 0.38, 0.28]

                      相应的，记录标签为：

                      labels = ["P", "N", "P", "N", "N", "N"]

                      记阈值为 threshold，根据阈值和分类器输出的概率值，可以得到分类器的预测结果：

                      # 根据给定的阈值和分类器输出的概率值，计算分类结果
                      def calculate_predictions(threshold, probabilities):
                          predictions = []  # 字符序列，用于储存输出结果
                          for prob in probabilities:
                              if prob >= threshold: 
                                  predictions.append("P")  # 输出结果不小于阈值时判定为 P
                              else:
                                  predictions.append("N")  # 输出结果小于阈值时判定为 N
                          return predictions

                      根据分类器的预测结果，计算真阳率 TPR 和假阳率 FPR：

                      # 计算真阳性率 TPR 和假阳性率 FPR
                      def calculate_tpr_fpr(predictions, labels):
                          tp = sum([1 for p, l in zip(predictions, labels) if p == "P" and l == "P"])  # TP
                          fn = sum([1 for p, l in zip(predictions, labels) if p == "N" and l == "P"])  # FN
                          fp = sum([1 for p, l in zip(predictions, labels) if p == "P" and l == "N"])  # FP
                          tn = sum([1 for p, l in zip(predictions, labels) if p == "N" and l == "N"])  # TN
                      

                          tpr = tp / (tp + fn)  # 真阳率
                          fpr = fp / (fp + tn)  # 假阳率
                      

                          return tpr, fpr

                      其中，计算规则参考：

                      (5)TPR=TPN+FPR=FPN−

                      T 4

                      软间隔 SVM 分类器通过引入松弛变量 ξi≥0 来放松优化约束，允许在分类中出现错误。

                      原始形式的软间隔 SVM 分类器如下：

                      (11)minw,b12||w||22+C∑i=1Nξis.t. yi(wTxi+b)≥1−ξi,i=1,...,Nξi≥0,i=1,...,N

                      以下说法是否正确？并给出理由。

                      T 4.1

                      增加超参数 C 倾向于减少训练误差。

                      正确。

                      超参数 C 越大，对误分类的惩罚越强，进而倾向于减少训练误差。

                       

                      T 4.2

                      增加超参数 C 往往会降低间隔。

                      正确。

                      增加超参数 C 意味着对误分类的惩罚增强，进而使模型更加强调正确分类样本的重要性，倾向于选择更小的间隔来让误分类的样本尽可能少，因此会降低间隔。

                       

                      T 4.3

                      硬间隔 SVM 是超参数 C 设置为 0 的软间隔的特殊情况。

                      错误。

                      硬间隔 SVM 要求训练数据是线性可分的，而软间隔 SVM 则允许一些特异点存在，采用对每个样本点 (xi,yi) 引入松弛变量 ξi 的方法放宽约束条件，此时要求超参数 C 是严格大于 0 的。

                       

                      T 4.4

                      增加超参数 C 往往会降低对异常值的敏感性。

                      错误。

                      超参数 C 越小，可能会使误分类样本数越多，进而使模型更容易受到异常值的影响；然而当超参数过大时，可能会导致模型过拟合，进而对异常值过度敏感。只有当超参数能较好起到平衡作用时，才能较好地降低对异常值的敏感性。

                       

                      T 5

                      考虑以下 4 个由不同核函数和/或松弛惩罚导致的 SVM 决策边界 (a) - (d)。标签 yi∈{−1,1} 的两类训练数据分别用三角形和正方形表示。支持向量被描述为实心三角形和正方形。请将每个决策边界与下面 (A) - (D) 中最可能的优化设置相匹配，并简要说明配对的理由。

                      (A) C=10−1 的软间隔线性 SVM。

                      (B) C=102 的软间隔线性 SVM。

                      (C) 核函数为 K(x,z)=exp⁡(−10−1||x−z||2) 的硬间隔 SVM。

                      (D) 核函数为 K(x,z)=exp⁡(−102||x−z||2) 的硬间隔 SVM。

                      首先，可以看出图 (a) 和 (c) 对应线性 SVM，图 (b) 和 (d) 对应非线性 SVM。

                      对于图 (a) 和 (c) ，由于增大超参数 C 会增强模型对误分类样本的惩罚力度，使模型倾向于选择更小的间隔来减少误分类样本，进而导致决策边界更为严格，使一些原本位于决策边界附近的样本点变成非支持向量。因此，支持向量较少的图 (a) 应该对应超参数较大的优化设置 (B)，支持向量较多的图 (c) 应该对应超参数较大的优化设置 (A)。

                      对于图 (b) 和 (d)，采用高斯核函数处理非线性可分的样本分类问题，高斯核函数可以表示为：

                      (12)K(x,z)=exp⁡(−||x−z||22σ2)

                      其中，方差 σ2 决定了高斯核函数的峰值尖锐程度，较小的 σ2 会使高斯核函数的峰值更加尖锐，样本之间的相似度衰减的更快，进而使模型更加关注每个样本的局部细节；与此相反，较大的 σ2 会更加关注样本的整体特征，得出更为平滑的决策边界。因此，决策边界较为平滑的图 (b) 应该对应 σ2 较大的优化设置 (C)，稍显过拟合的图 (d) 则应该对应 σ2 较小的优化设置 (D)。
